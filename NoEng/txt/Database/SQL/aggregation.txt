### Aggregation :
COUNT - count of not null
```roomsql
SELECT COUNT(column_name)
FROM table_name
WHERE condition;
```
SUM - sum of values
```roomsql
SELECT SUM(column_name)
FROM table_name
WHERE condition;
```
AVG - average
```roomsql
SELECT AVG(column_name)
FROM table_name
WHERE condition;
```
MIN - min value
```roomsql
SELECT MIN(Price)
FROM Products;
```
MAX - max value
```roomsql
SELECT MAX(Price)
FROM Products;
```
