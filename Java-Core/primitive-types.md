### Primitive type

- byte from -128 to 127  (8bit)
- short from --32,768 to 32,767  (16bit)
- int from -2(31) to 2(31) -1 (32bit)
- long from -2(63) to 2(63)-1  (64bit)
- float (32bit)
- double (64bit)
- char (16bit)
- boolean
 
- BigDecimal - class 
For that, you will need to use the java.math.BigDecimal class instead. Numbers and Strings covers BigDecimal and other useful classes provided by the Java platform.
